# Основные статистические функции в Python

## Пакет `pandas`

|Метод `pandas`|Описание|
|-|-|
|`.read_csv(filepath)`|Импорт данных из файла в формате CSV|
|`.read_excel(filepath)`|Импорт данных из файла в формате MS Excel|

|Метод `pandas.DataFrame`|Описание|
|-|-|
|`.describe()`|Основные описательные статистики|
|`.min()`|Минимальное значение|
|`.max()`|Максимальное значение|
|`.mean()`|Выборочное среднее|
|`.median()`|Выборочная медиана|
|`.var(ddof=0)`|Выборочная (смещённая) дисперсия|
|`.var(ddof=1)`|Выборочная (несмещённая) дисперсия|
|`.std(ddof=0)`|Выборочное (смещённое) стандартное отклонение|
|`.std(ddof=1)`|Выборочное (несмещённое) стандартное отклонение|

## Пакет `numpy`

|Метод `numpy`|Описание|
|-|-|
|`.min(a)`|Минимальное значение для массива `a`|
|`.max(a)`|Максимальное значение для массива `a`|
|`.mean(a)`|Выборочное среднее для массива `a`|
|`.median(a)`|Выборочная медиана для массива `a`|
|`.var(a, ddof=0)`|Выборочная (смещённая) дисперсия для массива `a`|
|`.var(a, ddof=1)`|Выборочная (несмещённая) дисперсия для массива `a`|
|`.std(a, ddof=0)`|Выборочное (смещённое) стандартное отклонение для массива `a`|
|`.std(a, ddof=1)`|Выборочное (несмещённое) стандартное отклонение для массива `a`|

## Пакет `scipy.stats`

|Метод|Описание|
|-|-|
|`.sem(a,ddof=1)`|Выборочное стандартное отклонение|
|`.ttest_1samp(a, popmean, alternative='two-sided')`|Одновыборочный t-тест на сравнение с `popmean`|
